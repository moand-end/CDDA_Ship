这是https://github.com/CleverRaven/Cataclysm-DDA/ 的与碧蓝航线的衍生mod

关于版本：
分为稳定版，和实验版
稳定版结构为：0.{游戏版本版号}.x.y，例如0.H.1.0
x的变化遵循每一次更新后无法兼容于上一个版本或出现大面积重构便会重置y版号并加一
例如0.H.1.14  —> 0.H.2.0
y的变化遵循每一次更新都能够兼容上一个版本的存档，并且没有无法修复的报错的情况下加一。
例如0.H.1.14 —>0.H.1.15
而每一次稳定版本的变更必定重置版号
例如：0.H.5.14 —>0.I.1.0
实验版结构为：e.{游戏版本版号}.{实验版日期}.x.y，例如:e.I.25-1-18.1.0
这里的x，y的变化也是如上所述。
而这里的实验版日期变更遵循汉化ckm合集日期的变更，以及本体模组有了不错的更新后，会酌情变更。
而变更实验版日期后，也会重置版本号

注意：在更新更新的版本，或者是实验版后，则不会对之前的版本进行任何游戏性的提升，除非以往版本出现报错之类的，才会进行打补丁更新。
